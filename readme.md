Présentation de l’application

Cette application est une solution web de gestion destinée aux centres/boutiques, développée avec Angular pour le frontend et Node.js / Express avec MongoDB pour le backend.

Elle permet :

🔐 L’authentification sécurisée des utilisateurs

👤 La gestion des profils (administrateur, gestionnaire, etc.)

🛍️ La gestion des produits et du stock

🛒 La gestion des ventes et des commandes

📊 Le suivi des performances via un tableau de bord dynamique

🔔 La gestion des notifications

L’objectif principal est de digitaliser et optimiser la gestion administrative et commerciale du centre à travers une interface moderne, responsive et sécurisée.
